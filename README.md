# Naver Blog Helper (Manual Workflow)

네이버 블로그 운영을 보조하는 **수동 승인형** 크롬 확장 프로그램입니다.

> 이 확장 프로그램은 댓글 등록/서로이웃 신청을 자동으로 실행하지 않습니다.
> 실제 상호작용(등록/신청 클릭)은 사용자가 직접 수행해야 합니다.

## 기능

- 키워드 입력 후 네이버 블로그 검색 페이지 열기
- 댓글 템플릿 저장 및 클립보드 복사
- 검색 결과 페이지에서 "다음 포스팅" 열기
- 포스팅 화면에서 댓글/이웃 관련 버튼 강조 표시

## 설치 방법

1. 이 폴더를 로컬에 저장합니다.
2. 크롬 주소창에 `chrome://extensions` 입력
3. 우측 상단 "개발자 모드" 활성화
4. "압축해제된 확장 프로그램을 로드합니다" 클릭
5. 이 폴더(`/workspace/hp`)를 선택

## 사용 방법

1. 확장 팝업에서 키워드를 입력하고 `네이버 블로그 검색 열기` 클릭
2. 검색 결과 탭에서 `검색 결과에서 다음 포스팅 열기` 버튼으로 포스팅 순차 열기
3. 포스팅 페이지에서 `현재 글에서 댓글/서로이웃 버튼 강조` 클릭
4. 저장한 댓글 템플릿을 `템플릿 복사`로 복사 후, 직접 붙여넣어 댓글 작성

## 주의

- 서비스 약관 및 운영 정책을 준수하세요.
- 무분별한 반복 댓글/신청은 계정 제한 사유가 될 수 있습니다.
